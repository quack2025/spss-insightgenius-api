Frequently Asked Questions

Do I need to install SPSS to use this? expand_more

No. InsightGenius reads .sav, .zsav, .csv, and .xlsx files natively. You don't need any local software installed. Everything happens in the cloud or via our API.

Is my data safe? expand_more

Security is our priority. For guest users, files are automatically purged from our servers 30 minutes after processing. All data is encrypted in transit and at rest.

How does the Claude integration work? expand_more

We provide a Model Context Protocol (MCP) server. You can add it to your Claude Desktop or Projects. Claude then calls InsightGenius's engine to perform statistical tests on your behalf.

Can I automate this with Zapier? expand_more

Yes. Our Business and Enterprise plans include full API access, which can be triggered via Zapier, Make.com, or custom code to create report pipelines.

What statistical tests are supported? expand_more

We support Independent T-Tests for means and Z-Tests for proportions, with customizable confidence levels (80%, 90%, 95%, 99%). We also handle ANOVA and correlation analysis.

When is the full web app launching? expand_more

The full project-management suite is scheduled for Q2 2026. Join our waitlist to get early beta access and founding member pricing.

How do you compare to Displayr? expand_more

Displayr is a great full-stack dashboard tool. InsightGenius is built for "interstitial" workflows—the quick analysis you need to do inside your existing tools like Claude or Excel, without moving your whole project to a new platform.
